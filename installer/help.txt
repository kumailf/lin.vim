Usage: ./install [OPTIONS]

  Install with 3 modes: full (default), limit and basic.

Notice:
  In full mode, you could use '--disable-xxx' options to disable specific feature.
  The '--disable-highlight --disable-color --disable-language --disable-editing' option is equivalent to '--limit'.
  The '--disable-color' option is conflict with '--static-color'.

Options:
  -b, --basic              Basic mode
  -l, --limit              Limit mode
  -s, --static-color TEXT  Use static colorscheme instead of random selections
  --disable-color          Disable extra colors such as colorschemes and
                           random selections
  --disable-highlight      Disable extra highlights such as RGB and cursor
                           word
  --disable-language       Disable coc language server extensions
  --disable-editing        Disable extra editing enhancement
  --disable-ctrl-keys      Disable windows ctrl+{a,s,x,c,v} keys
  --disable-plugin TEXT    Disable specific vim plugins in org/repo format
                           from github
  --disable-vim            Only support neovim, disable vim
  --disable-neovim         Only support vim, disable neovim
  --help                   Show this message and exit.
